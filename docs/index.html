Project Timeline

Design Phase Complete

ERD v1–v11 (840 tables), Architecture Review (30 sections), Function Library (148 functions), Dev Plan, Business Strategy. All frozen.

Cycle 0: Platform Foundation Complete

Monorepo, NestJS + Next.js + Prisma, Docker, 29 platform tables, IAM (444 permissions), tenant isolation, auth, CI/CD.

Cycles 1–3: Core Teaching & Learning Complete + Reviewed

SIS, attendance, classroom, grading, communications, notifications. ADR-057 envelope. Content moderation. 52 tenant tables, 60 endpoints, 7 consumers.

Cycles 4–5: Staff Operations Complete + Reviewed

HR (employees, leave, certifications, compliance), Scheduling (timetable, calendar, coverage, substitutes). Identity mapping resolved. 27 tenant tables, 53 endpoints, 2 consumers.

Cycle 6 + 6.1: Enrollment, Payments & Profile Complete + Reviewed

Admissions pipeline, family billing (Stripe stubbed), self-service profile editing, household management. Full parent journey end-to-end. ~23 tables, ~41 endpoints, 2 consumers.

Parent Testing Pause Complete

Validated the parent experience. Shipped improvements: add-child workflow with admin approval, calendar RSVP, open enrollment search with geolocation, multi-school billing, dashboard icon redesign.

Cycle 7: Tasks & Approval Workflows Next

Cross-cutting infrastructure. Task Worker (ADR-011) auto-creates tasks from Kafka events. Workflow Engine (ADR-012) handles multi-step approvals. Generalizes leave/absence/child-link patterns. 12 tables.
